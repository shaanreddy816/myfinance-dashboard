   - Customize and download icons
   - Save as `icon-192.png` and `icon-512.png`
   - Place in `famledgerai/public/` folder

2. **Deploy** (1 min):
   ```bash

   git add .
   git commit -m "Add PWA support for Android and iOS"
   git push origin main
   ```

## 🆘 Troubleshooting

### "Add to Home Screen" not showing?
- **Android**: Use Chrome (not other browsers)
- **iOS**: Use Safari (not Chrome)
- Clear browser cache
- Check HTTPS is working

### Icon not showing?
- Verify files are in `/public/` folder
- Check file names match exactly
- Clear cache and reinstall

### App not working offline?
- Visit all pages once while online
- Wait 2-3 minutes for caching
- Check browser console for errors

**See `TESTING_GUIDE.md` for complete troubleshooting**
